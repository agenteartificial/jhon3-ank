# Templo Espiritista - Landing Page

## LOG DE MUTACIÓN APLICADO
Se han anulado las instrucciones "Dark Luxury, Flip Cards, Plus Jakarta Sans" y se ha aplicado la siguiente identidad visual aleatoria por orden del usuario:

- **Vibra Visual:** [Opción C] **Organic Earth** (Fondo oscuro con tonos tierra `bg-primary: #1C1917`, colores principales rojo rubí `#BE123C`, texturas de ruido sutil `noise-overlay` y formas totalmente asimétricas/orgánicas).
- **Tipografía:** [Opción B] **Playfair Display** para títulos (Display) y **Lora** para cuerpo de texto.
- **Layout:** [Opción B] **Zig-Zag Storytelling** para los servicios en vez de Flip Cards. Alternando imágenes orgánicas a la izquierda y derecha.
